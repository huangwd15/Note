# 装饰器

本质是传递函数。

```python
import time
def show_time(func):
     def wrapper():
         start_time = time.time()
         func()
         end_time = time.time()
         print 'spend %s' % (end_time - start_time)
     return wrapper
 
@show_time
 def foo():
     print 'hello'
     time.sleep(3)
 
foo()

hello
spend 3.00506687164
```

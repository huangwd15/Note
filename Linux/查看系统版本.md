# 查看系统版本

- `lsb_release -a`

```
LSB Version:    :base-4.0-amd64:base-4.0-noarch:core-4.0-amd64:core-4.0-noarch
Distributor ID: CentOS
Description:    CentOS release 6.9 (Final)
Release:        6.9
Codename:       Final
```

- `cat /etc/issue`

```
CentOS release 6.9 (Final)
Kernel \r on an \m
```

两条命令都适合所有Linux发行版
